# HayvanatBahcesiOdev

N11 Java Bootcamp kapsamında hazırlanmış OOP ödevidir.  
Bu projede hayvanat bahçesi yönetimi, polymorphism (çok biçimlilik) kullanılarak modellenmiştir.

##  Açıklama
Hayvanların ortak özellikleri:
- Tür adı
- Ağırlık
- Yaş

Her hayvan için:
- `getDosage()`
- `getFeedSchedule()`

metotları tanımlanmış ve alt sınıflarda override edilerek farklı davranışlar sağlanmıştır.

##  Kullanılan Yapılar
- Abstract Class
- Inheritance
- Polymorphism
- Method Overriding
